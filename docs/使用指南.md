# 使用指南

本文档提供详细的操作步骤和命令说明。

## 环境准备

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 下载模型

```bash
# 方式1：从 ModelScope 下载（国内推荐）
python scripts/download_model.py \
    --model_name Qwen/Qwen2.5-4B-Instruct \
    --source modelscope \
    --save_dir ./models

# 方式2：从 Hugging Face 下载
python scripts/download_model.py \
    --model_name Qwen/Qwen2.5-4B-Instruct \
    --source huggingface \
    --save_dir ./models
```

### 3. 准备数据

```bash
# 下载数据集
python scripts/download_data.py

# 或创建示例数据（用于快速测试）
python scripts/download_data.py --create_sample

# 预处理数据
python scripts/preprocess_data.py \
    --raw_data ./data/raw/medical_qa.json \
    --output_dir ./data/processed \
    --max_samples 15000

# 准备不同规模数据集（1k/5k/10k）
python scripts/prepare_data_splits.py \
    --train_file ./data/processed/train.json \
    --output_dir ./data/processed
```

## 训练模型

### LoRA 微调

```bash
# LoRA 1k
python train.py --config configs/lora_1k.yaml

# LoRA 5k
python train.py --config configs/lora_5k.yaml

# LoRA 10k
python train.py --config configs/lora_10k.yaml
```

### QLoRA 微调

```bash
# QLoRA 1k
python train.py --config configs/qlora_1k.yaml

# QLoRA 5k
python train.py --config configs/qlora_5k.yaml

# QLoRA 10k
python train.py --config configs/qlora_10k.yaml
```

### 恢复训练

```bash
python train.py \
    --config configs/lora_1k.yaml \
    --resume_from_checkpoint outputs/lora_1k/checkpoint-500
```

## 配置文件说明

### LoRA 配置示例

```yaml
# configs/lora_1k.yaml
model_name_or_path: "Qwen/Qwen2.5-4B-Instruct"

lora_config:
  r: 8                    # LoRA rank
  lora_alpha: 32          # LoRA alpha
  target_modules:         # 目标模块
    - "q_proj"
    - "k_proj"
    - "v_proj"
    - "o_proj"
  lora_dropout: 0.1

training_args:
  output_dir: "./outputs/lora_1k"
  num_train_epochs: 3
  per_device_train_batch_size: 4
  learning_rate: 1.0e-4

data_config:
  train_file: "./data/processed/train_1k.json"
  validation_file: "./data/processed/dev.json"
  test_file: "./data/processed/test.json"
```

### QLoRA 配置示例

```yaml
# configs/qlora_1k.yaml
model_name_or_path: "Qwen/Qwen2.5-4B-Instruct"

quantization_config:
  load_in_4bit: true
  bnb_4bit_compute_dtype: "float16"
  bnb_4bit_use_double_quant: true
  bnb_4bit_quant_type: "nf4"

lora_config:
  r: 8
  lora_alpha: 32
  target_modules:
    - "q_proj"
    - "k_proj"
    - "v_proj"
    - "o_proj"

training_args:
  output_dir: "./outputs/qlora_1k"
  per_device_train_batch_size: 8  # QLoRA 可以用更大的 batch size
```

## 常见问题

### Q: 显存不足怎么办？

**A**: 
1. 使用 QLoRA（4-bit 量化）
2. 减小 `per_device_train_batch_size`
3. 增大 `gradient_accumulation_steps`
4. 减小 `max_source_length` 和 `max_target_length`

### Q: 训练速度慢怎么办？

**A**:
1. 使用更小的数据集测试（1k）
2. 减少 `num_train_epochs`
3. 启用 `fp16=true`
4. 使用更好的 GPU

### Q: 如何查看训练日志？

**A**:
```bash
# 使用 TensorBoard
tensorboard --logdir outputs/lora_1k/logs

# 或查看日志文件
cat outputs/lora_1k/logs/events.out.tfevents.*
```

### Q: 模型保存在哪里？

**A**:
- 训练过程中的检查点：`outputs/lora_1k/checkpoint-{step}/`
- 最佳模型：`outputs/lora_1k/checkpoint-best/`
- 最终模型：`outputs/lora_1k/`

### Q: 如何修改超参数？

**A**: 编辑对应的配置文件（如 `configs/lora_1k.yaml`），修改 `training_args` 部分
